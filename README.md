#   Webオンライン対戦型ダウト大富豪
## 1. 作品概要
Webブラウザ上で動作する、オンライン対戦機能付きの大富豪ゲームです。
※学校部活内でのグループ制作のプロジェクトの一部であり、自身が担当したゲームロジック部分を公開しています。

## 2. 担当範囲
- **担当:** ゲーム進行管理、カード提出制限ロジック
- **使用言語:** HTML(index.html),JavaScript (index.js)
- **主な実装機能:**
    - サーバとの通信(リアルタイム更新)
    - 場のカードに対して数値・枚数の大小比較

## 3. 技術的なこだわり
### 速度重視の処理
作品展示の機会が学校の文化祭で、QRコードを用意してお客様にプレイしてもらう形だったためリアルタイム性を重視した。フロントエンドからの不正な操作をサーバで監視しリスクを減らす形ではなく、フロントエンドで役やターンを即座に判断しサーバに共有する処理を選んだ。
